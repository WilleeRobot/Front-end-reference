Overview

This guide is a simple reference tool created to serve as a memory aid for learning and solidifying front-end development concepts. It will be continuously updated as time allows and hopefully grow to include discussions, observations, and learnings about other front-end development frameworks.

HTML

HTML (Hypertext Markup Language) is the basic building block of any webpage. It defines the content and structure of any webpage. Other languages that are typically used in front-end web development such as JavaScript and CSS (below) define the overall appearance, style, and interactivity of the page itself. The hypertext part of the language refers to links that connect web pages to one another. This can be either internal links or external links. The element in HTML is what provides instruction for any given piece of content on the page. These elements are setoff from other pieces of text on the page via ‘tags’.

CSS

Cascading style sheets (CSS) is code that defines styles applied to a website. It is not considered to be a language or even markup. Rather it uses Selectors and Declarations to define the styles to be applied to HTML elements. Declarations are in turn comprised of a property and a property value. Things that can be styled are plenty. Examples include but are not limited to color, size, text size, font properties, shapes, and more. Layouts can also be uniquely fine-tuned to cause the webpage to look and feel according to how you like. Columns can be created in addition to scrolling boxes, floating elements, and more.

CSS - Layout

CSS layout techniques allow us to take elements in a webpage and control their positioning relative to the following factors: it’s default position in a normal layout, other elements on the page, it’s parent container, and the overall viewport. The primary method of controlling the layout of elements is by using the display property.

The table below summarizes the most commonly used property values associated with the display property.

Common property values of the display property

Property value What it does
inline Changes the display of the element to appear beside its neighboring elements, like words in a paragraph
block Changes the display of the element to appear one below another.
in-line block Changes the dispay of the element to appear inline as above but also maintains block-level customizability for things like height, width, margins, padding etc.
